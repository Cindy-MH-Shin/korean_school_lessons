-고 나서 순서

-아/어서 연속동작 (하다 → 해서)

⚠️ -고 vs -아/어서 — 헷갈리는 차이!
🔵 -고 = 그냥 순서/나열 (neutral list/sequence) — 인과관계 없어요! → 걷고 쉬어요. (walked AND rested)
🟢 -아/어서 = 원인→결과 또는 꼭 연속동작 → 걸어서 피곤해요. (walked, THEREFORE tired)
✨ -아/어서 앞에는 절대 과거형 -았/었 금지!  ❌ 뛰었어서 → ✅ 뛰어서 피곤해요.

⭐ 빠른 참고 Quick Reference — 세 가지 연결어미

💡 핵심 규칙 정리! (Key Rules Summary)

① -고 = and / then → 어간 + 고 · 불규칙 없음 · 시제 자유 · 동사 & 형용사 모두 OK
   → 걷고 쉬어요. / 예쁘고 착해요.
② -아/어서 = so / because / and then → 모음조화 ⚠️ · 앞 절에 시제 금지 ⚠️
   → 뛰어서 피곤해요. / 날씨가 좋아서 걸어요.
③ -지만 = but / however → 어간 + 지만 · 불규칙 없음 · 시제 자유 · 일상에서 매우 자주!
   → 걷고 싶지만 피곤해요. / 비가 오지만 걸었어요.
⚠️ 절대 금지! -아/어서 앞에 -았/었 ❌ → ❌ 뛰었어서 → ✅ 뛰어서 피곤해요.
⚠️ 주의! -지만 뒤에 그러나/하지만 중복 금지 ❌ → ❌ 피곤하지만 그러나 → ✅ 피곤하지만

Mnemonic: Blue -고 = neutral "and" (no emotion, just listing) · Teal -아/어서 = "therefore" (cause causes result) · Purple -지만 = "nevertheless" (contrast, like English "but"). The #1 student error: using -아/어서 for past-tense first clauses — always resist the urge to add -었!
🔵 -고 핵심
나열/순서: 뛰고 걸어요.
동시: 음악을 듣고 걸어요.
형용사: 예쁘고 착해요.
과거 가능: 뛰었고 쉬었어요. ✅

No vowel harmony · No irregulars · Most flexible!
🟢 -아/어서 핵심
이유: 뛰어서 피곤해요.
순서(꼭): 가서 걸었어요.
ㅏ/ㅗ → 아서: 좋아서, 가서
나머지 → 어서: 뛰어서, 먹어서
하다 → 해서: 피곤해서

⚠️ NO -았/었 before -아/어서!
🟣 -지만 핵심
대조: 피곤하지만 행복해요.
소망+역접: 걷고 싶지만 바빠요.
과거 가능: 뛰었지만 안 빨랐어요.

No vowel harmony · Very common in daily speech · Never add 그러나/하지만 after!
❌ 흔한 실수 Common Mistakes
❌ 뛰었어서 → ✅ 뛰어서
❌ 걷어서 → ✅ 걸어서 (ㄷ→ㄹ!)
❌ 피곤하지만 그러나 → ✅ 피곤하지만
❌ 걷고서 피곤해요 → ✅ 걸어서 피곤해요

-고 is NEVER used for cause/reason!
🌿 복합 문장 연습 Combined Sentence Practice
✨ 세 가지 어미 모두 합쳐서! — Using all three in one sentence:

많이 걸어서 피곤하지만 행복해요!  (I walked a lot, so I'm tired, but I'm happy!)
