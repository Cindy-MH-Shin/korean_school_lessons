📌 꼭 기억하세요! (Key Points to Remember!)

✅ 규칙 동사 (Regular verbs): 어간(Stem) + 는 게 어때요?
    사다 → 사- + 는 게 어때요 = 사는 게 어때요?
    먹다 → 먹- + 는 게 어때요 = 먹는 게 어때요?

⚠️ ㄹ 탈락 동사 (ㄹ-drop verbs): 받침 ㄹ이 ㄴ 앞에서 사라져요! (ㄹ disappears before ㄴ!)
    열다 → 여- + 는 게 어때요 = 여는 게 어때요?
    들다 → 드- + 는 게 어때요 = 드는 게 어때요?
    만들다 → 만드- + 는 게 어때요 = 만드는 게 어때요?

ㄹ-drop rule: When a verb stem ends in ㄹ as a final consonant (받침), the ㄹ disappears whenever the following syllable starts with ㄴ. Since ~는 게 어때요 begins with ㄴ (는), the ㄹ always drops: 열다 → 여는, 들다 → 드는, 만들다 → 만드는.
